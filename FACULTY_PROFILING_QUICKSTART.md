# Faculty Profiling - Quick Start Guide

## 🚀 5-Minute Setup

### **Step 1: Access Feature** (30 seconds)
1. Login to SchedulaPro
2. Click **"Faculty Profiling"** in sidebar (UserCog icon)

### **Step 2: Select Faculty** (30 seconds)
1. Search or browse faculty list
2. Click **"Create Profile"** or **"Edit Profile"**

### **Step 3: Set Essentials** (3 minutes)

#### Tab 1: Courses (60 seconds)
- Slide to rate 3-5 top preferred courses (7-10)
- Leave others at 0

#### Tab 2: Time (60 seconds)
- **Toggle** 3-4 preferred days
- **Check** morning or afternoon preference
- **Add** any unavailable time slots (meetings, etc.)

#### Tab 3: Workload (30 seconds)
- Set **Min hours**: e.g., 12
- Set **Max hours**: e.g., 24
- Set **Preferred days/week**: e.g., 3-4

#### Tab 4: Constraints (30 seconds)
- Set **Max daily hours**: e.g., 6
- Toggle **No back-to-back** if needed
- Set **Min break**: e.g., 15-30 minutes

### **Step 4: Save** (10 seconds)
Click **"Save Profile"** ✅

## 🎯 What the AI Uses

### **HARD Constraints** (Never Violate)
- ❌ Unavailable time slots

### **SOFT Constraints** (Prefer to Satisfy)
- ✅ Course preferences (weighted)
- ✅ Preferred days
- ✅ Time preferences (morning/afternoon)
- ✅ Workload range
- ✅ Daily/consecutive hour limits
- ✅ Break requirements
- ✅ Room/building preferences

## 💡 Quick Tips

### **DO's** ✅
- ✅ Mark **all unavailability** accurately
- ✅ Rate **top 5 courses** you want to teach
- ✅ Set **realistic** workload ranges
- ✅ Update profiles **before optimization**
- ✅ Use **unavailable slots** for critical conflicts

### **DON'Ts** ❌
- ❌ Don't be too restrictive (causes conflicts)
- ❌ Don't leave profile empty (AI can't help)
- ❌ Don't forget to save changes
- ❌ Don't set impossible constraints
- ❌ Don't mark everything as "preferred"
