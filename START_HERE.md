## 📦 What You're Deploying

**SchedulaPro** is a complete classroom scheduling system with:

- ✨ **Frontend**: React app hosted on Vercel (worldwide CDN)
- 🗄️ **Backend**: Supabase database + API (PostgreSQL + Edge Functions)
- 🔐 **Authentication**: Secure login with role-based access
- 📊 **Features**: Drag-drop scheduling, approvals, reports, analytics
